🚀 Nova Connect
Nova Connect is a unified placement management platform designed to bridge the gap between students and recruiters. Unlike general professional networks, it focuses on streamlined profile management, skill tracking, and AI-driven candidate filtering to make the hiring process more efficient.

✨ Key Features
Student Dashboard: Manage profiles, track technical skills, and receive real-time job notifications.

Recruiter Portal: Advanced candidate filtering based on specific tech stacks and performance metrics.

AI Integration: Powered by Gemini AI for intelligent skill classification and an interactive support chatbot.

Data Visualization: Clear insights for both students and HRs to track placement progress.

🛠️ Tech Stack
Frontend: React.js (Vite), TypeScript, Tailwind CSS

Backend: Node.js, Express.js

Database: MongoDB / MySQL

AI: Google Gemini API

⚙️ Installation & Setup
Follow these steps to run the project locally:

Clone the repository:

Bash
git clone https://github.com/nandanavijesh/nova-connect-81-main.git
cd nova-connect-81-main
Install dependencies:

Bash
npm install
Set up Environment Variables:
Create a .env file in the root directory and add your credentials (API keys, DB strings):

Code snippet
VITE_GEMINI_API_KEY=your_key_here
DATABASE_URL=your_db_url
Run the application:

Bash
npm run dev
The app will be available at http://localhost:5173.
